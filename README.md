# BOLTEST Frontend

A modern React frontend for test case management and TFS sync.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Project Structure

- `public/` — Static assets
- `src/` — Source code (components, pages, services, hooks, utils, styles)
- `.env.example` — Environment variable example

## Tech Stack
- React 18
- React Router DOM
- Axios
- Tailwind CSS

## API
- Set your backend API URL in `.env` or `.env.local` using `REACT_APP_API_URL`.
